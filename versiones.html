Versiones TypeScript

Actualmente TypeScript esta en la version 4.6.4, publicada el 28 de abril del 2022. En la sigueinte tabla se puede encontrar las versiones major de TS, donde si se pulsa sobre cada Version, se le redigira a la página donde se explica con detalle los cambios de cada version.

Version Mes/Año Descripcion de las versiones
V1.0 04/2014 TypeScript empieza a ser mantenida por Microsoft
V2.0 09/2016 Simplificacion en la instalacion, tipos de flujo de control analizados y añadido 'readonly'
V3.0 07/2018 Expansión en el tipo Tuplas, añadido 'unknow' como tipo
V4.0 10/2020 Tipos de tuplas variadas, indicacion de si un metodo esta obsoleto
