Diferencias entre TypeScript y JavaScript

TypeScript incluye varias mejoras frente al uso con JavaScript. A continuación se muestran las mas importantes:

  • Sistema de tipos: añade tipos básicos como, ademas de String y Number:
    1. Boolean: dato logico que reprenta verdadero o falso
    2. Array: dato estructurado que permite almacenar una colección de elementos
    3. Tuple: similar al array, pero con un número fijo de elementos escritos
    4. Enum: representa al tipo enumeración
    5. Any: indica que la variable puede ser de cualquier tipo. Es muy útil a la hora de trabajar con librerías externas
    6. Void: indica que una función no devolverá ningún valor
    7. Never: ste tipo representa el tipo de valores que nunca se producen
  • TS ofrece comprobación de tipos
  • Codigo en TypeScript tiene que ser compilado mientras que JavaScript no
  • TypeScript ofrece un soporte directo para modulos, mientras que JavaScript lo hace a traves de ECMAScript 6
  • TypeScript permite el uso de las interfaces, dando asi la posibilidad de crear escenarios más avanzados, en JavaScript no hay soporte para las interfaces
